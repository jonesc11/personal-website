How'd I start?

In seventh grade, I played a lot of Minecraft. I bought my own Minecraft server and wanted to modify my server to my exact specifications

So naturally, I began making plugins for my server using Java. These plugins handled player interactions with everything in the server, and opened my eyes to object-oriented programming. In high school, I took more classes in object-oriented programming and refined my skills from my self-learned base.

Since then, I started school at Rensselaer Polytechnic Institute, and I'm now studying Computer Science as one of my majors, and I've had multiple projects since then...

Projects

Solving Paint by Pairs

In C++ using recursion, I wrote a program to find all possible solutions to a game called Paint by Pairs. This was a homework assignment for my Data Structures course that I took in the fall semester of 2016. As part of this assignment, I optimized my program to solve the puzzles quicker, and I found ways to minimize the amount of memory I used.

Likeable Local

In the spring of 2016, I participated in an internship with Likeable Local. The thing that I worked the most on was using Python to pull data from Facebook's interests and likes and determine whether or not certain interests would be useful for advertising. For example, if I searched "Law", "Lawyer" would be useful, but not "Jennifer Lawrence."
