Projects

RateMyCourses

RateMyCourses is a tool for students to use to decide which courses to take based on other students' past experiences. Students could enter the website to read ratings and reviews on courses for any university, as well as post their own ratings and reviews. This was built in HTML, CSS, JavaScript, AJAX, and PHP. Although I touched on all aspects of the project, my main focus was on the PHP and JavaScript/AJAX. This was a project for Web Systems Development in the Fall of 2017.

Paint By Pairs Solver

In C++ using recursion, I wrote a program to find all possible solutions to a game called Paint by Pairs. As part of this assignment, I optimized my program to solve the puzzles quicker, and I found ways to minimize the amount of memory I used. This was a homework assignment for Data Structures in the Fall of 2016.

RPI Fixxer

RPI Fixxer is an online ticketing system for the school's maintenance team that users can log into via the school's authentication service and submit repair requests. Workers can then view that request, claim the request, and produce status updates on that job. Admins could see what jobs were taken by which workers to track productivity. This was a project for Introduction to ITWS in the Fall of 2016.
